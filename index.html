双分支工作流

Created2025-10-07Updated2025-10-07

设置步骤（在第一台电脑上初始化）：

假设你的仓库名为 username.github.io，且已经可以通过 hexo d 部署。

  1. 初始化本地 Git 仓库并切换分支
    在你的 Hexo 博客根目录下（这里有 _config.yml, source, themes 等文件夹），打开终端。

    # 初始化 Git
git init

# 创建一个专门存放源文件的分支，比如叫 hexo-source
git checkout -b hexo-source

# 将本地仓库关联到你的 GitHub 远程仓库
git remote add origin https://github.com/yourusername/yourusername.github.io.git

    请将 yourusername 替换为你自己的 GitHub 用户名。

  2. 提交并推送源文件

    # 添加所有文件到暂存区
git add .

# 提交更改
git commit -m "Initial commit of Hexo source files"

# 将 hexo-source 分支推送到 GitHub
git push origin hexo-source

    现在，你的所有源文件都已经安全地备份在 GitHub 的 hexo-source 分支上了。

在第二台电脑上克隆并工作：

现在，你换到另一台电脑，想要继续写博客。

  1. 克隆源文件分支

    # 克隆你刚刚创建的源文件分支
git clone -b hexo-source https://github.com/yourusername/yourusername.github.io.git my-blog
cd my-blog

  2. 安装依赖
    由于 node_modules 被忽略了，你需要重新安装 Hexo 运行所需的环境。

    npm install
# 或者如果你使用了 hexo-cli
# npm install hexo-cli -g
# npm install

  3. 正常写作和部署

    # 写新文章等...
hexo new "我的新文章"

# 生成、部署
hexo clean && hexo g && hexo d

# 将源文件的更改也推送到 GitHub
git add .
git commit -m "Publish new post: 我的新文章"
git push origin hexo-source

日常同步流程：

从此以后，你在任何一台电脑上工作后，都需要执行两条命令：

  1. git push origin hexo-source (推送源文件更改)
  2. hexo d (部署静态网站)

在开始工作前，最好先执行一次 git pull origin hexo-source 来拉取另一台电脑可能做的更新。

外星人秘密交流工具

Created2025-09-23Updated2025-10-07

噜啦咯咧

点我开始使用秘密对话

源码

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Text Encoder/Decoder</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            max-width: 600px;
            margin: 0 auto;
            padding: 20px;
        }
        textarea {
            width: 100%;
            height: 100px;
            margin-bottom: 10px;
        }
        button {
            padding: 10px;
            margin-right: 10px;
        }
        #result {
            margin-top: 20px;
            padding: 10px;
            border: 1px solid #ccc;
            min-height: 100px;
        }
    </style>
</head>
<body>
    <h1>Text Encoder/Decoder</h1>
    <textarea id="inputText" placeholder="Enter text here..."></textarea>
    <div>
        <button onclick="encodeText()">Encode</button>
        <button onclick="decodeText()">Decode</button>
    </div>
    <div id="result"></div>

    <script>
        function encodeText() {
            const input = document.getElementById('inputText').value;
            if (!input) {
                alert('请输入需要加密的文本！');
                return;
            }

            // 将中文字符转换为UTF-16编码（4位十六进制）
            let hexString = '';
            for (let i = 0; i < input.length; i++) {
                const code = input.charCodeAt(i);
                hexString += code.toString(16).padStart(4, '0');
            }

            // 严格映射十六进制字符到“啦咯噜咧”
            const map = {
                '0': '咯咯', '1': '咯啦', '2': '咯噜', '3': '咯咧',
                '4': '啦咯', '5': '啦啦', '6': '啦噜', '7': '啦咧',
                '8': '噜咯', '9': '噜啦', 'a': '噜噜', 'b': '噜咧',
                'c': '咧咯', 'd': '咧啦', 'e': '咧噜', 'f': '咧咧'
            };

            let encodedText = '';
            for (const char of hexString) {
                encodedText += map[char] || '';
            }

            document.getElementById('result').textContent = encodedText;
        }

        function decodeText() {
            const input = document.getElementById('inputText').value;
            if (!input) {
                alert('请输入需要解密的文本！');
                return;
            }

            // 检查输入是否仅包含“啦”、“咯”、“噜”、“咧”
            if (!/^[啦咯噜咧]+$/.test(input)) {
                alert('加密内容包含非法字符！');
                return;
            }

            // 确保输入长度为偶数
            if (input.length % 2 !== 0) {
                alert('加密内容长度不正确！');
                return;
            }

            // 反向映射“啦咯噜咧”到十六进制字符
            const map = {
                '咯咯': '0', '咯啦': '1', '咯噜': '2', '咯咧': '3',
                '啦咯': '4', '啦啦': '5', '啦噜': '6', '啦咧': '7',
                '噜咯': '8', '噜啦': '9', '噜噜': 'a', '噜咧': 'b',
                '咧咯': 'c', '咧啦': 'd', '咧噜': 'e', '咧咧': 'f'
            };

            // 将加密字符串转换为十六进制
            let hexString = '';
            for (let i = 0; i < input.length; i += 2) {
                const pair = input.substr(i, 2);
                hexString += map[pair] || '';
            }

            // 将十六进制字符串还原为原始文本
            let decodedText = '';
            for (let i = 0; i < hexString.length; i += 4) {
                const hex = hexString.substr(i, 4);
                if (hex.length === 4) {
                    const charCode = parseInt(hex, 16);
                    if (!isNaN(charCode)) {
                        decodedText += String.fromCharCode(charCode);
                    } else {
                        console.error(`Invalid hex code: ${hex}`);
                    }
                }
            }

            document.getElementById('result').textContent = decodedText;
        }
    </script>
</body>
</html>

一个简单的文本编码器，采用了 UTF-16 编码原理用“啦咯噜咧”这四个字代替了十六进制数字
